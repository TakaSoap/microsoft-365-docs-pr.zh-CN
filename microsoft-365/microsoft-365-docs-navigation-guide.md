## <a name="breadcrumbs"></a>痕迹导航

痕迹导航位于目录的标题下方和上方，并指示当前文章在目录下的位置。  这不仅有助于将上下文设置为正在阅读的内容类型，还允许您在目录树上向后导航：

![Microsoft 365痕迹导航。](media/m365-breadcrumb.png)
