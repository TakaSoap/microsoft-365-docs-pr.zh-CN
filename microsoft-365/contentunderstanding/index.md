# <a name="introduction-to-microsoft-sharepoint-syntex"></a>Microsoft SharePoint Syntex 简介

SharePoint Syntex 是一项 Microsoft 365 服务，可帮助组织：

- 使用先进的 AI 和机器教学来丰富人类体验
- 自动化内容处理
- 将内容转换为知识

此页面上的资源旨在帮助你开始学习和使用组织中的 SharePoint Syntex。
