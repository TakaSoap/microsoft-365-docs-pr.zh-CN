# <a name="how-microsoft-identifies-malware-and-potentially-unwanted-applications"></a>Microsoft 如何识别恶意软件和可能不需要的应用程序

Microsoft 致力于通过努力Windows安全并控制设备，提供令人愉悦且高效的用户体验。 Microsoft 通过识别和分析软件与联机内容来帮助防止潜在威胁。 下载、安装和运行软件时，我们将检查已下载程序的信誉，并确保你免受已知威胁的侵害。 对于我们未知的软件，您也会受到警告。  

你可以提交未知 [或可疑的软件进行分析，以帮助](https://www.microsoft.com/wdsi/filesubmission/) Microsoft。 这将有助于确保我们的系统扫描未知或可疑软件以开始建立信誉。 [详细了解如何提交文件进行分析](submission-guide.md)

以下各节概述了我们用于应用程序的分类以及导致该分类的行为类型。

>[!NOTE]
> 正在快速开发并分发新的恶意软件形式和可能不需要的应用程序。 以下列表可能并不全面，Microsoft 保留在不提前通知或通知的情况下调整、扩展和更新这些列表的权利。

## <a name="unknown--unrecognized-software"></a>未知 – 无法识别的软件  

没有完善的防病毒或保护技术。 识别并阻止恶意站点和应用程序，或信任新发布的程序和证书需要花费一些时间。Internet 上近 20 亿个网站和软件持续更新和发布，不可能包含有关每个网站和计划的信息。

将未知/不常见的下载警告视为可能未检测到的恶意软件的早期警告系统。 通常，新恶意软件的发布时间会一直延迟到识别。 并非所有不常见的程序都是恶意程序，但对于普通用户而言，未知类别中的风险要高很多。 不阻止未知软件的警告。 如果需要，用户可以选择正常下载并运行应用程序。

一旦收集了足够的数据，Microsoft 的安全解决方案就可以做出决定。 找不到任何威胁，或者应用程序或软件被归类为恶意软件或可能不需要的软件。

## <a name="malware"></a>恶意软件

恶意软件是应用程序和其他代码（如软件）的总体名称，Microsoft 更具体地将这些代码分类为 *恶意软件* 或 *不需要的软件*。

### <a name="malicious-software"></a>恶意软件

恶意软件是危及用户安全的应用程序或代码。 恶意软件可能会窃取个人信息、锁定设备，直到支付勒索金额、使用设备发送垃圾邮件或下载其他恶意软件。 通常，恶意软件想要欺骗、欺诈或欺骗用户，使用户处于易受攻击状态。

Microsoft 将大多数恶意软件分为以下类别之一：

* **后门：** 一种恶意软件，可允许恶意黑客远程访问和控制你的设备。

* **命令和控件：** 一种恶意软件，可感染你的设备并建立与黑客的命令和控制服务器的通信，以接收指令。 建立通信后，黑客可以发送可以窃取数据、关闭和重新启动设备以及中断 Web 服务的命令。

* **下载者：** 一种将其他恶意软件下载到设备的恶意软件。 它必须连接到 Internet 以下载文件。

* **放置器：** 一种将其他恶意软件文件安装到设备的恶意软件。与下载程序不同，放置器不必连接到 Internet 来放置恶意文件。 放置的文件通常嵌入在放置器本身中。

* **攻击：** 一段代码，使用软件漏洞获取对设备的访问权限并执行其他任务，如安装恶意软件。 [请参阅有关攻击详细信息](exploits-malware.md)。

* **Hacktool：** 可用于获取对设备未经授权的访问的工具类型。

* **宏病毒：** 一种通过受感染的文档传播的恶意软件，例如Microsoft Word或Excel文档。 病毒在您打开受感染的文档时运行。

* **模糊器：** 一种隐藏其代码和用途的恶意软件，使安全软件更难检测或删除。

* **密码窃取程序：** 收集个人信息（如用户名和密码）的恶意软件类型。 它通常与键记录器一起运行，它可以收集并发送有关你按下的键和所访问的网站的信息。

* **勒索软件：** 一种恶意软件，可加密文件或进行其他修改，以阻止你使用设备。 然后，它会显示勒索说明，指出你必须先付款或执行其他操作，然后才能再次使用设备。 [请参阅有关勒索软件详细信息](/security/compass/human-operated-ransomware)。

* **未授权安全软件：** 冒充安全软件但不提供任何保护的恶意软件。 这种类型的恶意软件通常显示有关设备上不存在的威胁的警报。 它还尝试诱使你支付其服务费用。

* **特洛伊木马：** 一种尝试看似无害的恶意软件。 与病毒或蠕虫不同，特木马本身不会传播。 相反，它会尝试尝试合法地欺骗用户下载和安装它。 安装后，特洛伊木马会执行各种恶意活动，如窃取个人信息、下载其他恶意软件或向攻击者提供对设备的访问权限。

* **特洛伊木马单击器：** 自动单击网站或应用程序上的按钮或类似控件的特洛伊木马类型。 攻击者可以使用此特洛伊木马单击在线广告。 这些单击可能会扭曲在线投票或其他跟踪系统，甚至可以在设备上安装应用程序。

* **蠕虫：** 一种传播到其他设备的恶意软件。 蠕虫可以通过电子邮件、即时消息、文件共享平台、社交网络、网络共享和可移动驱动器传播。 复杂的蠕虫利用软件漏洞传播。

### <a name="unwanted-software"></a>不需要的软件

Microsoft 认为你应该可以控制你的Windows体验。 在设备上Windows软件应让你通过明智的选择和可访问的控件来控制你的设备。 Microsoft 识别可确保你保持控制的软件行为。 我们将未完全证明这些行为的软件分类为"不需要的软件"。

#### <a name="lack-of-choice"></a>缺少选择

必须通知你设备上发生了什么，包括哪些软件有效以及它是否处于活动状态。

出现缺乏选择的软件可能：

* 无法提供有关软件行为及其目的和意图的醒目通知。

* 无法清楚地指示软件何时处于活动状态。 它也可能尝试隐藏或伪装其状态。

* 未经你的许可、交互或同意，安装、重新安装或删除软件。

* 安装其他软件，而不明确指示其与主软件的关系。

* 避开浏览器或操作系统中的用户同意对话框。

* 假地声称是 Microsoft 的软件。

软件不得误导或强迫你做出有关设备的决策。 它被认为是限制你的选择的行为。 除了上一列表之外，缺乏选择的软件可能：

* 显示有关设备运行状况的过度声明。

* 对设备上的文件、注册表项或其他项进行令人误解或不准确的声明。

* 以一种有关设备运行状况的友好方式显示声明，并需要付款或某些操作来交换解决声称的问题。

存储或传输活动或数据的软件必须：

* 通知你并同意这样做。 软件不应包含将其配置为隐藏与存储或传输数据相关的活动的选项。

#### <a name="lack-of-control"></a>缺乏控制

必须能够控制设备上的软件。 你必须能够启动、停止或以其他方式撤销对软件的授权。

缺乏控制的软件可能：

* 阻止或限制您查看或修改浏览器功能或设置。

* 在未授权的情况下打开浏览器窗口。

* 重定向 Web 流量，而不发出通知和征得同意。

* 未经你的同意，修改或操作网页内容。

更改浏览体验的软件只能使用浏览器支持的扩展模型进行安装、执行、禁用或删除。 不提供支持的扩展模型的浏览器被视为不可扩展且不应修改。

#### <a name="installation-and-removal"></a>安装和删除

你必须能够启动、停止或以其他方式撤销对软件的授权。 软件应在安装前征得你的同意，并且必须提供一种清晰明了的方式来安装、卸载或禁用该软件。

提供较差安装 *体验的软件* 可能会捆绑或下载 Microsoft 分类的其他"不需要的软件"。

提供较差删除 *体验的软件* 可能：

* 尝试卸载时出现令人困惑或令人误解的提示或弹出窗口。

* 使用标准安装/卸载功能失败，如"添加/删除程序"。

#### <a name="advertising-and-advertisements"></a>广告和广告

在软件本身之外推广产品或服务的软件可能会干扰您的计算体验。 在安装显示广告的软件时，你应该有明确的选择和控制。

软件呈现的广告必须：

* 包括一种让用户关闭广告的明显方法。 关闭广告的行为不得打开另一个广告。

* 包括显示广告的软件的名称。

显示这些广告的软件必须：

* 使用软件所显示的广告中显示的相同名称，为软件提供标准卸载方法。

向用户显示的广告必须：

* 与网站内容可区分。

* 不要误解、误解或混淆。

* 不包含恶意代码。

* 不调用文件下载。

#### <a name="consumer-opinion"></a>消费者意见

Microsoft 维护分析员和智能系统的全球网络，您可以在 [其中提交软件进行分析](https://www.microsoft.com/wdsi/filesubmission)。 你的参与可帮助 Microsoft 快速识别新恶意软件。 分析后，Microsoft 会为满足所述条件的软件创建安全智能。 此安全智能将软件标识为恶意软件，并通过 Microsoft 反恶意软件解决方案Microsoft Defender 防病毒可供所有用户使用。

## <a name="potentially-unwanted-application-pua"></a>PUA (可能不需要) 

我们的 PUA 保护旨在保护用户工作效率并确保用户Windows体验。 此保护有助于提供更高效、更高效、更令人Windows体验。 若要了解如何在基于应用程序的 Chromium 和 Microsoft Edge Microsoft Defender 防病毒 PUA 保护，请参阅检测并阻止[可能不需要的应用程序](/microsoft-365/security/defender-endpoint/detect-block-potentially-unwanted-apps-microsoft-defender-antivirus)。

*PUA 不被视为恶意软件。*

Microsoft 使用特定类别和类别定义将软件分类为 PUA。

* **广告软件：** 显示广告或促销或提示您填写软件中除其本身外的其他产品或服务的调查的软件。 这包括向网页插入广告的软件。

* **Torrent software (Enterprise only)**： Software that is used to create or download torrents or other files specifically used with peer-to-peer file-sharing technologies.

* **加密软件 (Enterprise加密) ：** 使用你的设备资源来挖掘加密货币的软件。

* **捆绑软件：** 提供用于安装不是由同一实体开发或运行该软件不需要的其他软件的软件。 此外，根据本文档中列出的条件，提供用于安装其他符合 PUA 要求的软件的软件。

* **营销软件：** 监视用户活动并将其传输到应用程序或服务（而非自身）进行市场营销研究的软件。

* **安装软件：** 主动尝试规避安全产品检测的软件，包括存在安全产品时行为不同的软件。

* **较差的行业信誉：** 受信任的安全提供程序可检测其安全产品的软件。 安全行业致力于保护客户并改进他们的体验。 Microsoft 和安全行业其他组织不断交换我们分析的文件知识，以为用户提供可能的最佳保护。
