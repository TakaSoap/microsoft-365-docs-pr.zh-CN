- 安全管理员
- 安全操作员

## <a name="turn-on-preview-features"></a>启用预览功能

你有权访问即将推出的功能，可以在这些功能全面发布之前提供反馈以帮助改善整体体验。

启用预览体验设置，以率先体验即将推出的功能。

1. 在导航窗格中，选择“**设置**”。
2. 选择 **"Microsoft 365 Defender"。**
3. 选择“**预览功能**” > “**启用预览功能**”。 
4. 选择“**保存**”。

当看到 **启用预览功能**”复选框处于选中状态时，你会知道已开启了预览功能。
