Microsoft Defender 安全中心是一个门户，你可以在这里访问 Microsoft Defender for Endpoint 功能。

使用 **威胁&漏洞** 管理仪表板，可扩大组织整体安全状况的可见性。 你将看到需要关注的设备和建议，它们可以帮助你减少组织中的攻击面。

使用 **威胁分析仪表板** 持续评估和控制 Spectre 和 Spectre 的风险暴露。
