## <a name="what-you-need-to-know"></a>须知内容

使用公共预览版中的功能时，这些功能：

- 可能具有受限的功能。 例如，功能只适用于一个平台。
- 通常请在正式发布 (GA) 之前了解功能更改。
- 完全受 Microsoft 支持。
- 只能在所选地理区域或云环境中使用。 例如，功能可能在政府云中不存在。
- 预览中的各个功能可能有更多的使用情况和支持限制。 如果是，此信息通常会在功能文档中注明。
- 预览版本提供标准支持级别，可用于生产环境。

## <a name="turn-on-preview-features"></a>启用预览功能

你有权访问即将推出的功能，可以在这些功能全面发布之前提供反馈以帮助改善整体体验。

启用预览体验设置，以率先体验即将推出的功能。

1. 在导航窗格中，选择 **"设置** \>  \> **终结点高级功能** \> **预览功能"。**

2. 切换 **开和关** 之间的 **设置，** 然后选择 **保存首选项**。
