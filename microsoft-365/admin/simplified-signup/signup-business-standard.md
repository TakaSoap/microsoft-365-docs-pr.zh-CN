# <a name="sign-up-for-a-microsoft-365-business-standard-subscription"></a>注册 Microsoft 365 商业标准版订阅

想要注册Microsoft 365 商业标准版订阅？

可通过多种方式入门：

- **[购买](https://go.microsoft.com/fwlink/?linkid=2109654)Microsoft 365 商业标准版并完成自己的设置**。 若要购买Microsoft 365 商业标准版，[请执行以下步骤](#sign-up-steps)。
- **对于 Microsoft 合作伙伴**：如果你是合作伙伴，请参阅 ["获取Microsoft 365业务](../../business/get-microsoft-365-business.md)"。

## <a name="sign-up-steps"></a>注册步骤

若要注册和购买Microsoft 365 商业标准版，请完成以下步骤。

> [!IMPORTANT]
> 注册 Microsoft 365 商业版的人员（通常是业务所有者）自动成为组织的技术管理员。 如果希望帮助管理 Microsoft 365 服务，可以将其他人添加为管理员。 有关详细信息，请参阅[分配管理员角色](../add-users/assign-admin-roles.md)。

1. 在 ["业务Microsoft 365"页](https://go.microsoft.com/fwlink/?linkid=2109654)上，选择 **"查看计划&定价**。
2. 在下一页上，了解每月成本，然后向下滚动页面，了解有关Microsoft 365中包含的内容的详细信息。 在Microsoft 365 商业标准版下，选择 **"立即购买**"。
3. 在"**感谢你选择Microsoft 365 商业标准版** 页上，输入信息以开始使用。 选择 **下一步**。
4. 输入已使用的电子邮件地址。 这可以是你希望 Microsoft 在安装过程中用来与你通信的任何地址。 这也是我们将向你发送有关帐单和续订信息的地址。 然后选择 **"设置帐户**"。
5. 输入你的姓名、业务电话号码、业务规模、公司名称和位置。 选择 **下一步**。

   > [!NOTE]
   > 我们会在管理中心显示你的公司名称。 你可以在此处管理Microsoft 365用户、许可证和其他功能和服务。 我们还将其包含在任何内部SharePoint站点 URL 中。

6. 帮助我们确保这是你。 输入可用于联系你的数字，然后选择 **"发送验证码**"。 你将收到一条文本。 输入代码并选择 **"验证**"。

7. 确定如何登录到Microsoft 365。 可以通过添加域或[使用当前个人电子邮件登录](#option-1-sign-in-with-your-outlook-hotmail-yahoo-gmail-or-other-email-account)来[创建新的业务电子邮件帐户](#option-2-create-a-new-business-email-account-and-attach-a-domain)。

## <a name="terms-of-service-update-for-simplified-sign-up-mode"></a>简化注册模式的服务更新条款

**适用于：以前使用简化注册模式购买的Microsoft 365 商业标准版的现有订阅者**。

如果以前使用简化注册模式在 2021 年 10 月之前购买业务订阅，而不添加业务域，则可能需要接受新的服务条款，以便不间断的服务和使用Microsoft Office应用。 你可能会收到电子邮件，或者登录到Microsoft 365管理中心时会看到应用内提示。

简化的注册使用条款最近已更新，特别是它们阐明了多用户业务订阅的许可和数据所有权。 若要继续为业务订阅提供服务，可以访问 [Microsoft 365管理中心](https://go.microsoft.com/fwlink/?linkid=2024339)并继续使用 **简化注册** (并同意使用 Microsoft 服务协议补充) ，或访问 [Microsoft 365管理中心](https://go.microsoft.com/fwlink/?linkid=2024339)并添加 **业务域** (并使用 Microsoft Online Services 协议) 。 若要帮助你选择最适合你需求的这两种模式，请参阅本文顶部的表。

如果选择不接受简化注册或添加业务域的更新条款，则订阅不会自动续订，并且在当前订阅合同结束时，你将失去对Office应用的访问权限。 OneDrive数据将保留 90 天，供你复制，然后将其删除。

## <a name="frequently-asked-questions"></a>常见问题解答

### <a name="what-is-a-business-email-and-what-are-the-advantages-to-setting-one-up"></a>什么是商业电子邮件，设置电子邮件有哪些优势？

业务电子邮件是使用你自己的域名的电子邮件。 例如，如果你拥有域名 `contoso.com`，则可以使用 URL `www.contoso.com`生成网站，但也可以有自定义电子邮件地址，例如 yourname@contoso.com。 这被称为品牌商业电子邮件，因为它给你的电子邮件专业的外观。

### <a name="how-do-i-get-a-new-business-email-address"></a>如何实现获取新的商业电子邮件地址？

有三个选项可用于获取业务电子邮件。

- 可以免费使用建议的 onmicrosoft.com 域 (someone@mybusiness.onmicrosoft.com) 。
- 可以购买新域，以便 (mybusinessname@contoso.com) 提供更紧凑的电子邮件地址。
- 使用已拥有的域名。

### <a name="why-might-i-need-to-verify-my-domain-to-create-a-business-email"></a>为什么我可能需要验证我的域才能创建业务电子邮件？

如果选择使用已拥有的域，可以将其用于带Microsoft 365的电子邮件地址。 作为注册过程的一部分，我们要求你验证域，以便你可以通过Microsoft 365发送电子邮件。 这可确认你是使用该标识发送电子邮件的域的所有者，这可以增强安全性并防止欺诈活动。

### <a name="is-there-a-benefit-to-paying-monthly-vs-annually"></a>每月支付与每年支付费用是否有好处？

为了为客户提供最大的灵活性，可以使用不同的付款选项。

- Microsoft 365 商业基础版、适用于企业的应用、商业标准和业务高级版计划可用于每月承诺付款或年度承诺付款。
- 每月承诺付款：按月付款，可以随时取消。
- 年度承诺付款：注册一年订阅，但可以选择按月付款或在注册时支付全年费用。 使用此付款选项有折扣。

### <a name="how-does-recurring-billing-work"></a>定期计费的工作原理如何？

“定期计费”处于打开状态时，订阅将每年都在你的订阅之日继续计费。 如果你的订阅处于活动状态，你可以在管理中心中将其关闭或重新打开。 在 [关闭或开启轮次定期计费](../../commerce/subscriptions/renew-your-subscription.md#turn-recurring-billing-off-or-on)时了解详细信息。

### <a name="what-do-i-do-if-i-want-to-change-my-business-name"></a>如果要更改业务名称，该怎么办？

请联系我们的小型企业支持专家，他们可以帮助你更改业务名称。 了解有关 [获取支持的详细信息](../get-help-support.md)。
