# <a name="cortana-in-microsoft-365"></a>Microsoft 365中的Cortana

Cortana，你的个人生产力助理，提供 AI 支持的体验，以节省时间，并专注于最重要的事情。 Cortana旨在提供安全、安全地处理和推理电子邮件、文件、聊天等Office 365数据的功能，从而节省时间、提高效率并提高用户的工作效率。

使用有效的工作或学校帐户登录时，用户可以使用Cortana获得基于云的协助服务，Microsoft 365体验满足Office 365的企业级隐私、安全和合规性承诺 (“**Cortana企业服务**”) 。

- **Cortana企业服务包括** Windows 10 (版本 2004 及更高版本) 中的Cortana、适用于 iOS 和 Android 的Outlook、适用于 iOS 和 Android 的Microsoft Teams移动应用以及 [Microsoft Teams显示器](/microsoftteams/devices/teams-displays)。

- 这些不同的体验需要单独的许可条款，并具有下面所述的单独选择退出步骤。

- 与其他Office 365服务一致，Cortana企业服务符合在线[服务条款 (OST) ](https://www.microsoft.com/licensing/product-licensing/products)所反映的企业级隐私、安全和合规承诺。

- 新的Microsoft 365体验（如“简报电子邮件”和“播放我的电子邮件”）将使用Cortana企业服务启用，并完全遵守这些承诺。 这些功能目前在全球范围内提供 (标准多租户) 。 有关查找使用位置的详细信息，请访问 [查看帐户的其他属性值](../../enterprise/view-user-accounts-with-microsoft-365-powershell.md#view-additional-property-values-for-accounts)。

- 现有使用者体验（包括 Windows 10 (版本 1909 及更早) 中的Cortana）受 [Microsoft 服务协议](https://www.microsoft.com/licensing/product-licensing/products)和 [Microsoft 隐私声明](https://privacy.microsoft.com/privacystatement)的约束， (请参阅) 下面的“面向消费者的现有服务”部分。 这些条款还将控制使用使用者凭据登录时提供给用户的Cortana企业服务。

## <a name="what-data-is-processed-by-cortana-enterprise-services"></a>哪些数据由Cortana企业服务处理？

Cortana企业服务处理来自用户的查询、满足用户请求所需的Office数据，以及 Microsoft 系统为运行该服务而生成的其他遥测数据。 Cortana企业服务收集的数据包括用户口语查询的文本表示形式 (，即语音识别) 的听录。 此文本数据是客户数据，并根据  [联机服务条款](https://www.microsoft.com/licensing/product-licensing/products)进行管理。 它仅用于开发和改进与联机服务条款一致的机器学习模型。

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-enterprise-services"></a>Cortana企业服务中客户数据的治理模型是什么？

与其他Office 365服务一致，Cortana企业服务受联[机服务条款](https://www.microsoft.com/licensing/product-licensing/products)保护。 这包括一组承诺，用于保护客户数据免受意外丢失、更改、未经授权的披露或访问，或非法破坏。 客户数据还受到严格的访问限制。 Microsoft 仅使用客户数据来提供商定的服务，并用于与这些服务兼容的服务。 有关详细信息，请参阅下表。

## <a name="how-does-microsoft-store-retain-process-and-use-customer-data-in-cortana"></a>Microsoft 如何在Cortana中存储、保留、处理和使用客户数据？

下表介绍了Cortana企业服务的数据处理。

|名称|说明|
|---|---|
|**存储**|客户数据存储在Office 365云中的 Microsoft 服务器上。 数据是租户的一部分。 <br/><br/>语音音频不会保留。|
|**留在地理位置**|客户数据存储在地理位置Office 365云中的 Microsoft 服务器上。 数据是租户的一部分。|
|**保留**|当租户管理员关闭帐户或发出 GDPR 数据主体权限删除请求时，将删除客户数据。 <br/><br/>语音音频不会保留。|
|**处理和保密性**|参与处理客户数据和个人数据 (i) 的人员将仅根据客户的说明处理此类数据， (ii) 在交互结束后仍必须维护此类数据的机密性和安全性。|
|**使用情况**|Microsoft 仅使用客户数据来提供商定的服务，并用于与这些服务兼容的服务。 开发和改进模型的机器学习是其中一个目的。 机器学习是在Office 365云中完成的，不会有人查看、查看或标记客户数据。 <br/><br/>数据不用于面向广告。|

## <a name="cortana-enterprise-services-in-microsoft-365-experiences"></a>在Microsoft 365体验中Cortana企业服务

### <a name="cortana-in-windows-10-version-2004-and-later"></a>Windows 10 (版本 2004 及更高版本中的Cortana) 

Windows 10中的Cortana应用可帮助用户快速获取跨Microsoft 365的信息，使用类型化或口语查询与人员进行连接、检查日历、设置提醒等。

Cortana Windows可以帮助用户更好地管理其计划和任务。 他们可以在微软待办中添加到列表，查找本地信息，获取定义，并使用必应搜索跟踪最新新闻、天气和财务信息。

Windows 10版本 2004 及更高版本中Cortana符合Cortana企业服务的企业级隐私、安全性和合规性承诺，如[在线服务条款 (OST) ](https://www.microsoft.com/licensing/product-licensing/products)中所反映。

### <a name="how-to-opt-out-of-cortana-in-windows-10"></a>如何选择退出Cortana Windows 10

在Windows 10中选择退出Cortana不会选择退出单独控制的其他体验。 管理员可以使用 Experience\AllowCortana MDM 策略或通过 组策略：计算机配置\管理模板\Windows 10 Windows组件\Search\Allow Cortana 为组织配置Cortana。

从 Windows 10 版本 2004 开始，Cortana是预装了Windows的通用 Windows 平台 (UWP) 应用，通过Microsoft Store定期更新。 若要接收Cortana的最新更新，需要[通过Microsoft Store启用更新](/windows/configuration/stop-employees-from-using-microsoft-store)。

### <a name="play-my-emails"></a>播放我的电子邮件

通过Outlook移动) 连接到的“播放我的电子邮件” (是一种语音驱动的免提体验，用户可以通过手机、耳机或连接的音频设备通过扬声器收听其集中收件箱中的新邮件，并更改其一天。 用户可以要求Cortana大声朗读他们最近的电子邮件，并要求Cortana执行诸如标志、存档、删除和跳过邮件等操作。 此功能对于在上下班、多任务或外出时赶上电子邮件特别有用。 当用户在“播放我的电子邮件”中与Cortana交谈时，语音音频请求将直接发送到Cortana企业服务。 用户电子邮件的语音读出文本在Office 365云中处理。 在此过程中，不会在用户的移动设备上处理任何Office 365数据，并且不会保存任何电子邮件数据。 口述命令的脚本 (即“标记为已读”、“下一步”、“标志”等) 可根据 Microsoft [Online Services 条款](https://www.microsoft.com/licensing/product-licensing/products)中的数据保护条款保留。

Cortana会在电子邮件受到保护时发出，并在阅读邮件前短暂暂停，以给用户足够的时间暂停播放或跳到下一条邮件。 与私人电话通话类似，用户在可能无意中听到机密信息的位置启动播放时应小心谨慎。 在这些情况下，建议组织的员工在Outlook移动设备中使用“播放我的电子邮件”时，在适当的环境中佩戴耳机。

### <a name="how-to-opt-out-of-play-my-emails"></a>如何选择退出“播放我的电子邮件”

个人可以使用以下步骤选择退出“播放我的电子邮件”。

1. 打开Outlook移动设备。

2. 转到 **设置**。

3. 选择 **“播放我的电子邮件**”。

4. 将开关移到要禁用的帐户上。

[详细了解“播放我的电子邮件”](https://support.microsoft.com/help/4558256)

### <a name="briefing-email"></a>简报电子邮件

Cortana发送个性化的简报电子邮件，其中包含你所做的任务和承诺，并以方便的方式将它们标记为 **已完成** 或计划专注时间以完成这些任务。 它还包括会议摘要和当天的相关文档。 Cortana从用户的电子邮件中提取信息，并将其存储在其Exchange Online邮箱中，直到合并到“简报”电子邮件中。 在Exchange Online邮箱之外，任何时候都可以访问个人数据。 只有当用户拥有包含Exchange Online服务计划的许可证时，才能访问“简报”电子邮件。

### <a name="how-to-opt-out-of-briefing-email"></a>如何选择退出简报电子邮件

管理员可以在Exchange Online中使用 [PowerShell](/briefing/be-admin) 为组织配置简报。 个人可以通过在邮件页脚中选择 **“取消订阅**”，选择退出Cortana的简报电子邮件。

[详细了解简报电子邮件](https://support.microsoft.com/help/4558259)

## <a name="how-is-the-delivery-of-cortana-enterprise-services-different-from-the-delivery-of-other-cortana-features-i-may-have-previously-experienced"></a>Cortana企业服务的交付与我之前可能遇到的其他Cortana功能的交付有何不同？

下面是考虑企业中Cortana工作原理的两种方法：

**为具有Cortana企业服务的组织提供新体验**：Cortana企业服务旨在满足组织的安全性和合规性需求：

1. 这是一项新服务，本文档对此进行了讨论。

2. 对于受联机服务条款约束的服务，Microsoft 是一个数据处理器：Microsoft 从客户收集和使用客户数据，仅提供客户请求的联机服务以及客户指示的目的。 根据欧盟的《一般数据保护条例》 (GDPR) ，客户是其数据的数据控制者。 请参阅 [在线服务条款](https://www.microsoft.com/licensing/product-licensing/products) ，并 [介绍商业云客户的更多隐私透明度](https://blogs.microsoft.com/eupolicy/2019/11/18/introducing-privacy-transparency-commercial-cloud-customers/)。

3. 例如，“播放我的电子邮件”是一项Cortana服务，用户可以通过 iOS Outlook连接到该服务，并利用Cortana企业服务。

4. IT 管理员将始终控制Cortana的可选连接体验，类似于使用 Office ProPlus 应用程序时的可选连接体验。

**面向消费者的现有服务**：Cortana可选的连接服务主要针对使用者体验而设计，目前提供Windows 10 (版本 1909 及更早) 以及 iOS 和 Android 上的 Cortana 应用。

1. 这些体验支持天气、新闻和交通等功能。

2. 租户管理员可以控制Windows 10 (版本 1909 及更早) 中的Cortana以及 iOS 和 Android 上的Cortana应用是否能够允许Cortana连接到Office 365租户数据。

关闭Cortana对组织的 Microsoft 托管数据的访问

1. 在 <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">Microsoft 365 管理中心</a>中，选择 **设置** > **Org 设置**，然后选择 **Cortana**。

2. 取消选 **中Windows 10 (版本 1909 及更早) 中的“允许Cortana”复选框，以及 iOS 和 Android 上的Cortana应用，代表组织中的人员访问 Microsoft 托管的数据**，以禁用Cortana连接体验。

3. 选择“**保存更改**”。

对于受 [Microsoft 服务协议](https://go.microsoft.com/fwlink/p/?LinkId=2109174) 和 [Microsoft 隐私声明](https://privacy.microsoft.com/privacystatement)约束的服务，Microsoft 是数据控制器。 作为数据控制器，Microsoft 使用数据根据 [Microsoft 隐私声明](https://privacy.microsoft.com/privacystatement)改进产品和服务。
