# <a name="try-or-buy-a-microsoft-365-for-business-subscription"></a>尝试或购买业务订阅Microsoft 365

Microsoft 365企业版是一项订阅服务，可让你在 Microsoft 为你负责 IT 时在云中运行组织。 Microsoft 管理设备、防范实际威胁，并为组织提供最新的商业软件。 可以注册Microsoft 365 商业标准版、Microsoft 365 商业高级版或Microsoft 365 商业应用版的免费试用版订阅，并试用 30 天。

> [!NOTE]
> 注册免费试用版时，必须使用信用卡。 在免费试用期结束时，试用订阅会自动转换为付费订阅。 你的信用卡要到试用期结束才会计费。

::: moniker range="o365-21vianet"

> [!IMPORTANT]
> 不接受中国国际信用卡 **中由世纪互联运营的Office 365支付选项**。 您可以通过以下方式支付订阅费用：
>
> - 发票
> - 使用支付宝或中国银联支付证明的网上支付将以支付方式提供。 您可以在付款大约三 （3） 天后向 [发票系统提交发票](https://go.microsoft.com/fwlink/p/?LinkId=395314) 请求。 有关详细信息，请参阅 [申请由世纪互联运营的 Office 365 的发票](../admin/services-in-china/apply-for-a-fapiao.md)。

::: moniker-end

## <a name="before-you-begin"></a>准备工作

无需现有 Microsoft 帐户即可注册免费试用版。 对于本文中的所有其他过程，您必须是组织的全局或计费管理员。 有关详细信息，请参阅 [关于管理员角色](../admin/add-users/about-admin-roles.md)。

## <a name="try-a-free-trial-subscription"></a>试用免费试用版订阅

你是新客户，但还没有 Microsoft 帐户？ 使用以下步骤创建帐户并注册免费试用订阅Microsoft 365 商业标准版、Microsoft 365 商业高级版或Microsoft 365 商业应用版。

1. 转到<a href="https://www.aka.ms/office365signup" target="_blank">Microsoft 365产品网站</a>。
2. 选择要注册的计划，如 **Microsoft 365 商业标准版**，向下滚动页面，然后选择 **“免费试用”1 个月**。
3. 在下一页上，按照步骤设置帐户。
4. 注册过程可能需要几分钟才能完成。 \'完成后，即可为订阅启动安装向导。 有关设置订阅的详细信息，请参阅 [后续步骤](#next-steps)。

## <a name="buy-a-subscription-from-your-free-trial"></a>从免费试用版购买订阅

在免费试用期结束时，试用订阅会自动转换为付费订阅。 付费订阅默认为你当前拥有的计划。 可以按照“ [购买其他订阅](#buy-a-different-subscription)”中的步骤购买其他计划。

如果要在试用期结束前购买订阅，请执行以下步骤：

1. 在 Microsoft 365 管理中心中，转到“**计费**”\>“<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">你的产品</a>”页面。
2. 在 **“你的产品** ”页上，找到要购买的订阅。
3. 在 **“许可证”** 部分中，选择 **“购买订阅**”。
4. 选择订阅的每月或年度承诺，然后选择 **“签出**”。
5. 在下一页上，验证订阅，然后选择 **“签出**”。
6. 在下一页上，按 **此顺序** 验证 **已售地址**、**按信息计费** 和项。 如果需要进行任何更改，请选择适用部分旁边的 **“更改** ”。
7. \'完成后，选择 **“接受协议”&放置顺序**。

## <a name="extend-your-trial"></a>延长试用期

购买之前，是否需要更多时间来试用Microsoft 365的特色？ 如果试用版订阅将在 15 天内过期，并且之前尚未延长试用期，则试用期可以延长 30 天。 该操作仅可执行一次。

1. 在管理中心中，转到“**计费**”\>“<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">你的产品</a>”页面。
2. 在 **“产品”** 选项卡上，选择要扩展的试用订阅。
3. 在“订阅详细信息”页 **上的“订阅和付款设置”** 部分中，选择 **“延长结束日期**”。
4. 在 **“扩展结束日期”** 窗格中，查看扩展信息，并在必要时选择付款方式。 完成后，选择 **“扩展”试用版**。

准备好购买时，请参阅 [“购买试用版](#buy-a-subscription-from-your-free-trial)”。

## <a name="cancel-your-free-trial-subscription"></a>取消免费试用版订阅

如果决定在免费试用期结束前取消试用版订阅，请转到Microsoft 365 管理中心并[关闭定期计费](subscriptions/renew-your-subscription.md#turn-recurring-billing-off-or-on)。 当你的月份结束时，试用版将自动过期，你的信用卡将不会收费。

## <a name="try-a-different-subscription"></a>尝试其他订阅

如果已有业务订阅Microsoft 365，则可以使用Microsoft 365 管理中心来尝试其他订阅。

通过<a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">Microsoft 365 管理中心</a>]添加订阅时，新订阅与现有订阅 (域命名空间) 的组织关联。 通过此关联，可以更轻松地在订阅之间移动组织中的用户，或为其分配所需的其他产品的许可证。

1. 在管理中心，转到 **“计费** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">购买服务</a> ”页。
2. 在“ **购买服务** ”页上，你将看到组织可用的计划。 选择要尝试的Microsoft 365计划。
3. 在下一页上，选择 **“获取免费试用** 版”。 试用版提供 25 个用户许可证，为期一个月。
4. 选择接收短信或呼叫，输入电话号码，然后选择 **“给我发短信** ”或 **“给我打电话**”。
5. 输入验证码，然后选择 **“开始免费试用**”。
6. 在 **“签出** ”页上，选择 **“立即试用**”。
7. 在 **订单回执** 页上，选择 **“继续**”。

## <a name="buy-a-different-subscription"></a>购买其他订阅

如果已有业务订阅Microsoft 365，可通过Microsoft 365 管理中心为组织购买其他订阅。

### <a name="watch-move-users-to-a-different-subscription"></a>观看：将用户移动到其他订阅

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1SBE2?autoplay=false]

当用户更改角色时，他们可能需要当前Microsoft 365 商业高级版订阅中不可用的功能。 发生这种情况时，可以添加包含这些功能的新订阅，并向需要这些功能的人员分配许可证。

> [!NOTE]
> 对于某些订阅，只能在购买或续订订阅后的有限时间段内取消。 如果取消时段已过，请在订阅期限结束时关闭定期计费以取消订阅。

通过<a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">Microsoft 365 管理中心</a>购买另一个订阅时，新订阅与现有订阅 (域名空间) 的组织相关联。 这样可以更轻松地在订阅之间移动组织中的用户，或为其分配所需附加订阅的许可证。

1. 在管理中心，转到 **“计费** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">购买服务</a> ”页。
2. 在 **“购买服务** ”页上，选择要购买的计划，选择 **“详细信息**”，然后选择 **“购买**”。
3. 输入所需的许可证数，并选择是每月付款还是全年付款。 选择是否要自动将许可证分配给当前没有许可证的所有人。 然后 **立即选择“签出**”。
4. 查看定价信息，然后选择 **“下一步**”。
5. 提供付款信息，然后选择 **“就地订单** \> **转到管理主页**”。

> [!NOTE]
> 在试用版订阅过期后 90 天宽限期结束之前，必须将用户从免费试用版订阅移至新订阅。 通过执行此操作，可以保留数据、帐户和配置。 否则，将删除该信息。

::: moniker range="o365-21vianet"
