# <a name="order-microsoft-managed-desktop-devices"></a>设备Microsoft 托管桌面顺序

我们建议与批准的设备合作伙伴之一合作。 你可以与 Microsoft 帐户联系人合作，以了解有关建立设备合作关系的更多帮助。

Microsoft 托管桌面不再需要设备型号来自当前批准的[设备列表](../service-description/device-list.md)。 截至 2021 年 5 月 3 日，已批准制造商的所有设备都应满足我们发布硬件和软件要求。 你可以放心地继续使用该列表上的设备，但可以在购买商业Microsoft 托管桌面上找到Windows 10 专业版[推荐](https://www.microsoft.com/windowsforbusiness/view-all-devices)的设备。 在该网站中，通过展开"按筛选"区域中的功能查看推荐 **设备，** 然后选择 **"Microsoft 托管桌面"。** 每当计划首次在服务中注册特定设备型号时，都应该测试一个示例以确保它将提供你期望的用户体验。 有关详细信息，请参阅验证 [新设备](../get-started/validate-device.md)。
