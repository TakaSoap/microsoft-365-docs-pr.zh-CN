# <a name="work-with-processing-errors-in-advanced-ediscovery"></a>处理任务中的处理Advanced eDiscovery

*处理* 是文件标识、扩展嵌入文档和附件、文本提取以及图像文件的光学字符识别 (OCR) 以及后续索引该内容的过程。  

当您将保管人和非保管人数据源添加到"源"选项卡上的案例时，将处理来自 Microsoft 365 的所有部分索引项，使其完全可搜索。 同样，在将内容从源和非数据源Microsoft 365审阅Microsoft 365，也会处理此内容。

高级电子数据展示中的“**处理**”选项卡提供有关不同处理方案的高级索引编制状态的见解。
